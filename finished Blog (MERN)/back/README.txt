# Restful api

_It is an application to upload publications, it has user authentication_


### Instalation 🔧

_Download the project and run 'npm install' in your shell.
For development mode use 2 shells, in one use 'npm run ts', in the other use 'npm run dev'.
For user mode 'npm start'_



## Built with 🛠️

_NodeJS, Express, Multer, Bcrypt, Jsonwebtoken,Typescript, and MongoDB_
